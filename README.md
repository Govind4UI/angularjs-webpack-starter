# AngularJS - Webpack - Starter

A starter project using Angular 1.x with Webpack.

## Features

* ES6
* Sass
* Lazy load resources
* Hot Module Replacement


## Get Started

```
git clone https://github.com/robinskafte/angularjs-webpack-starter
cd angularjs-webpack-starter
npm install
npm run build
npm start
´´´
